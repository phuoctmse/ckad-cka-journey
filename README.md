# CKAD & CKA Journey (4 Months)

## 🎯 Mục tiêu
Kế hoạch học tập trong 4 tháng để chinh phục hai chứng chỉ Kubernetes:
- **CKAD** (Certified Kubernetes Application Developer)
- **CKA** (Certified Kubernetes Administrator)

## 📚 Tài liệu & Nguồn học
- [Docker Mastery](https://www.udemy.com/course/docker-mastery/)
- [KodeKloud](https://kodekloud.com/)
- [LabEx](https://labex.io/)

## 📈 Tiến trình học tập (Nhật ký hàng ngày)
Các báo cáo tiến độ (daily standup) của tôi sẽ được lưu trữ và quản lý trong thư mục `daily-logs/`.
Mỗi ngày một file markdown mới sẽ được tự động tạo dựa trên nội dung mẫu template.
